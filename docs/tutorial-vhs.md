## Narrative Scenarios (Dynamic GIFs)

Use these for "how it feels" walkthroughs.

### 1) Navigation and inspect loop

This flow demonstrates the core daily loop:
select a revision by item, inspect with `show`/`diff`, then return to context.

![Navigation and inspect loop](../target/vhs/tutorial-dynamic-navigation.gif)

### 2) Command mode, paging, and history

This flow shows exact command entry with `:`, paging behavior, and screen-history traversal.
It should feel familiar if you use shell history and terminal pagers often.

![Command mode, paging, and history](../target/vhs/tutorial-dynamic-command-history.gif)

### 3) Prompt and confirm safety flows

This flow demonstrates prompt and confirmation behavior for mutation commands.
It emphasizes safe defaults with explicit accept/reject interactions.

![Prompt and confirm safety flows](../target/vhs/tutorial-dynamic-safety.gif)

### 4) Remote and operation context

This flow covers remote sync and operation audit/recovery follow-through.
Use it to understand how fetch/push and operation history connect in one loop.

![Remote and operation context](../target/vhs/tutorial-dynamic-remote-ops.gif)
