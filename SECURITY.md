# Security Policy

## Supported Versions

This project is pre-1.0 and moves quickly. Security fixes are prioritized for the latest commit on
`main`.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

Please do not open public GitHub issues for suspected vulnerabilities.

Report security issues privately by emailing: `joshka@users.noreply.github.com`.

When reporting, include:

1. A clear description of the issue and impact.
1. Steps to reproduce (or a minimal proof of concept).
1. Affected versions or commit hashes.
1. Any suggested mitigations.

I will acknowledge reports within 72 hours and provide a status update after initial triage.
